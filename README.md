# Google Looker Studio - Sales Data Analysis

## Project Overview
Sales data analysis project using Google Looker Studio for an e-commerce platform covering 2021-2022 data.

## Tasks Completed
1. Monthly Revenue Trend (2022)
2. Sales by Payment Method
3. Average Order Value (AOV) Over Time (2022)
4. Total Revenue vs Discount Impact by Category
5. Top 10 Products with Largest Decrease in Sales (2021 vs 2022)
6. Weekend vs Weekday Sales in Q4 2022

## Tools Used
- Google Looker Studio
- Google Sheets
- Python (data preprocessing)

## Dataset
Contains e-commerce transactional data with fields:
- order_date, sku_name, category, payment_method
- before_discount, after_discount, qty_ordered
- year, month_id, day_name
